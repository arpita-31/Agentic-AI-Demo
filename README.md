# Agentic-AI-Demo
Planner AI 🚀

Planner AI is an Agentic AI-powered productivity assistant built using n8n, Google Gemini, Google Calendar, Google Sheets, and Telegram. It helps students and professionals automatically create study plans, manage deadlines, schedule events, and track progress through natural language conversations.

Features ✨
📚 AI Study Planner
Creates personalized study schedules based on:
Exam date
Subjects
Modules/topics
Available study hours
Breaks large modules into smaller study sessions.
Allocates more time to difficult topics.
Automatically includes revision sessions.
📅 Google Calendar Integration
Creates study sessions directly in Google Calendar.
Schedules webinars, workshops, meetings, and other events.
Provides a centralized view of all planned activities.
📊 Google Sheets Task Tracking
Stores all generated study tasks.
Tracks:
Subject
Module
Topic
Date
Time
Status
🤖 Intelligent Intent Detection

Phoenix Planner AI understands different user intents such as:

Create Study Plan
Add Webinar/Event
Complete Task
View Progress (future enhancement)
💬 Telegram Bot Integration

Users can interact with the assistant through Telegram using natural language:

Examples:

Create a DSA study plan for my exam on 15 September.
I have an AWS webinar tomorrow from 7 PM to 8:30 PM.
I completed Arrays.
🔄 Agentic Workflow

The system automatically:

Understands user intent.
Generates a plan or event.
Updates Google Sheets.
Creates Google Calendar events.
Sends confirmation back to Telegram.
Tech Stack 🛠️
n8n (Workflow Automation)

Telegram Bot API

Project Goal

Planner AI aims to act as a personal productivity assistant that not only creates plans but also manages schedules, tracks progress, and helps users stay on track toward their goals through intelligent automation and AI-driven decision making.
